Tutorial for Evgen and MCfast; bphyslib version

Since the reorganization of the directory structure, the bphyslib area now has the same directory structure as the product. The only difference is the command needed to setup the pacakages:

source ~bphyslib/setup_files/mcrel

This command will setup the development version of MCFast.

To select some other version, "source ~bphyslib/setup_files/mcrel version".

This command can be placed in your .login file.

You can now return to the main tutorial.
